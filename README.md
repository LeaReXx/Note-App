# Note-App

This is a note app written in Javascript without using any framework.

## Features:white_check_mark:

- Add a description for notes
- Change text and background color
- Change font size of  title and description
- See preview of note before save
- Dark theme for save your eyes
- Full Responsive

## Next Features:soon:
- Local Storage
- Date and time the note was made
